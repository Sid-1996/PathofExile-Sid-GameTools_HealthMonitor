## 🚀 項目簡介

PathofExile Sid GameTools_HealthMonitor 是一款功能完整的遊戲輔助工具，專為提升 Path of Exile 遊戲體驗而設計。通過實時監控角色狀態，自動執行各種輔助動作，讓玩家能夠更專注於遊戲策略。

### ✨ 核心特色

- 🔍 **智能血魔監控**：先進的顏色識別算法，精確檢測血量魔力狀態
- ⚡ **技能連段系統**：自訂觸發鍵和連段序列，自動執行技能組合
- 🎒 **一鍵清包功能**：智能背包物品檢測，自動清空指定物品
- 🖱️ **自動連點工具**：支援AHK腳本，實現自動滑鼠連點
- 🎯 **多重觸發設定**：支援多個觸發點設定，靈活應對不同情況
- 👁️ **即時預覽功能**：即時預覽監控區域，所見即所得
- ⚙️ **高度客製化**：豐富的設定選項，滿足不同玩家需求
- 🔍 **環境檢查工具**：自動檢查系統依賴項和環境配置
- 📋 **授權資訊系統**：功能分級和狀態顯示
- 🔄 **版本檢查功能**：自動更新檢測

## 📦 快速安裝

### 方法一：一鍵安裝（推薦）
1. 下載 [最新版本](releases/latest)
2. 解壓到任意目錄
3. 運行 `環境安裝.bat` 自動配置環境
4. 運行 `啟動工具.bat` 開始使用

### 方法二：開發環境設定
```bash
# 克隆倉庫
git clone https://github.com/Sid-1996/PathofExile-Sid-GameTools_HealthMonitor.git
cd PathofExile-Sid-GameTools_HealthMonitor

# 安裝依賴
pip install -r requirements.txt

# 運行示例代碼
python examples/color_detection_demo.py
python examples/screen_monitoring_demo.py
```

## 🎯 使用指南

### 主要功能模組

#### 🩸 血魔監控系統
1. **視窗選擇**：從下拉選單選擇遊戲視窗
2. **區域設定**：框選血量和魔力條區域
3. **觸發設定**：設定血量/魔力百分比和對應按鍵
4. **啟動監控**：開始即時監控和自動觸發

#### ⚡ 技能連段系統
1. **觸發鍵設定**：選擇任意按鍵作為連段觸發鍵
2. **序列設定**：設定1-5個技能按鍵作為連段組合
3. **延遲控制**：調整每個連段之間的延遲時間
4. **啟動系統**：開始背景監聽和自動執行

#### 🎒 一鍵清包功能
1. **UI截圖**：記錄背包介面截圖
2. **顏色基準**：設定淨空時的背包顏色
3. **自動偵測**：系統自動識別有物品的格子
4. **安全執行**：F3快速清空指定物品

#### 🖱️ 自動連點工具
1. **腳本選擇**：支援AHK腳本和編譯版
2. **權限設定**：自動以管理員權限啟動
3. **熱鍵控制**：CTRL+左鍵啟動自動連點
4. **安全退出**：程式關閉時自動停止

### 進階設定
- **顏色調整**：調整HSV參數以適應不同顯示設備
- **性能優化**：調整更新頻率和預覽設定
- **多配置管理**：為不同角色創建專屬配置
- **環境檢查**：驗證所有系統依賴項

詳細使用說明請參考：[使用說明文檔](docs/使用說明.md)
