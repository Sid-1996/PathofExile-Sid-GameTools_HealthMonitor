# Path of Exile 遊戲輔助工具 - 使用指南

本指南說明 v1.0.1 使用方式（單一 EXE 版本，免安裝）。

---

## 📦 下載與啟動

1. 到 Releases 下載 `GameTools_HealthMonitor_v1.0.1.zip`
2. 解壓縮到任意資料夾（建議非系統桌面根目錄）
3. 執行 `GameTools_HealthMonitor.exe`
4. 建議以系統管理員模式執行（避免熱鍵或截圖權限問題）

目錄常見檔案：

| 檔案 | 用途 |
|------|------|
| GameTools_HealthMonitor.exe | 主程式（所有功能整合） |
| auto_click.exe | 自動連點輔助（由主程式調用） |
| 使用說明.md | 精簡說明文件（可替換/刪除） |
| 啟動工具.bat | 可選啟動腳本 |

---

## ⏱️ 快速 60 秒初始化

| 步驟 | 動作 | 目的 |
|------|------|------|
| 1 | 啟動程式 | 進入主介面 |
| 2 | 設定血量區域 | 框選紅色血條區塊 |
| 3 | 設定魔力區域 | 框選藍色魔力條區塊 |
| 4 | 設定觸發值 | 例如：80%、50%、30% |
| 5 | 截取背包 UI | 為一鍵清包準備 |
| 6 | 設定 5 個取物座標 | 完成後可用 F6 |
| 7 | 測試熱鍵 | F3 / F6 / F10 驗證 |

---

## ⌨️ 熱鍵對照

| 熱鍵 | 功能 | 備註 |
|------|------|------|
| F3 | 一鍵清包 | 需先截圖 + 記錄淨空顏色 |
| F5 | 快速回藏身處 | 條件：角色可回程 |
| F6 | 一鍵取物 | 需設定 5 點座標 |
| F9 | 全域暫停 | 再按一次恢復 |
| F10 | 開/關血魔監控 | 需先設定區域 |
| F12 | 關閉工具 | 安全退出 |
| Ctrl+左鍵 | 自動連點 | 再按切換停止 |

---

## 🩸 血/魔監控設定

| 項目 | 操作 |
|------|------|
| 框選區域 | 按框選按鈕 → 拖曳選取 → 確認 |
| 觸發百分比 | 填入多組（例：80 / 50 / 30） |
| 冷卻控制 | 防止過度觸發（依內建邏輯） |
| 啟用/停止 | F10 開關 |
| 準確度建議 | 選純色區域，避免動態特效 |

---

## ⚡ 技能連段

| 項目 | 操作 |
|------|------|
| 建立套組 | 新增一組 → 設觸發鍵 |
| 定義順序 | 輸入按鍵序列（例：1,2,3,4） |
| 延遲設定 | 每步毫秒間隔（例：150ms） |
| 啟用/停用 | 勾選或取消該套組 |
| 測試 | 在安全區域按下觸發鍵 |

---

## 🎒 一鍵清包

| 項目 | 操作 |
|------|------|
| 截取背包 UI | 背包開啟 → 按按鈕截取 |
| 記錄淨空顏色 | 清空背包 → 記錄基準 |
| 執行 | 按 F3 |
| 動作邏輯 | 逐格判斷顏色差異後點擊 |
| 失敗排查 | 背包被遮擋 / 背景顏色不一致 |

---

## 📦 一鍵取物 (F6)

| 項目 | 操作 |
|------|------|
| 進入設定 | 按「設定取物座標」 |
| 依提示操作 | 將滑鼠移至位置 → Enter 確認 |
| 支援點數 | 固定 5 個座標 |
| 使用 | 於遊戲內按 F6 |
| 修改 | 重新執行設定流程覆蓋 |

---

## 🖱️ 自動連點

| 項目 | 操作 |
|------|------|
| 啟動 | Ctrl + 左鍵（按住一下） |
| 停止 | 再次 Ctrl + 左鍵 |
| 間隔 | 固定短延遲（不可自訂） |
| 適用場景 | 開箱、互動、連續採集 |

---

## ⏸️ 全域暫停

- 按下 F9 → 所有輸入/監控動作暫停（不觸發清包、取物、監控等）
- 再按一次恢復原狀態
- 適合聊天 / 暫時離開 / 測試環境

---

## 🛠️ 常見問題 (FAQ)

| 問題 | 解法 |
|------|------|
| 熱鍵失效 | 確認未暫停 (F9)；以管理員模式執行 |
| 監控不準 | 重框選，避開半透明與動態區域 |
| F3 沒反應 | 未先截取 UI 或未記錄淨空顏色 |
| F6 無動作 | 5 座標未設定完成或場景不符 |
| 自動連點沒啟動 | 防毒阻擋，加入白名單 |
| 工具卡住 | 關閉後重新啟動（設定會保存） |
| 防毒誤報 | 屬正常情況（鍵盤/截圖行為），加入信任 |

## ⚠️ 建議與限制
