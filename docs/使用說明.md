# PathofExile Sid 遊戲輔助工具 使用說明

## 📋 工具簡介

PathofExile Sid GameTools_HealthMonitor 是一款專為 Path of Exile（流亡黯道）遊戲設計的功能完整的自動化輔助工具。該工具整合了多種遊戲輔助功能，包括智能血魔監控、技能連段系統、一鍵清包功能、自動連點工具等，大幅提升遊戲體驗和操作效率。

## 🚀 快速開始

### 環境需求
- Windows 10/11 操作系統
- Python 3.10 或更高版本
- 以管理員權限運行（推薦）

### 安裝步驟
1. **環境安裝**：雙擊運行 `環境安裝.bat`，自動安裝Python和所需依賴包
2. **啟動工具**：雙擊運行 `啟動工具.bat` 或直接運行 `GameTools_HealthMonitor.exe`

## 🎮 主要功能模組

### 1. 🩸 血魔監控系統
- **即時檢測**：自動識別遊戲中的血量條和魔力條顏色變化
- **精確計算**：將血量和魔力轉換為0-100%的百分比顯示
- **視覺預覽**：提供即時的血量和魔力條預覽圖片
- **多重觸發**：支援設定多個血量/魔力觸發點
- **智慧冷卻**：防止過度觸發的冷卻時間機制

### 2. ⚡ 技能連段系統
- **觸發鍵設定**：選擇任意按鍵（QWERTYUIOPASDFGHJKLZXCVBNM）作為連段觸發鍵
- **序列組合**：設定1-5個技能按鍵作為連段組合
- **延遲控制**：每個連段之間可設定延遲時間（100-500毫秒）
- **多套組支援**：可同時啟用多個連段套組
- **背景運行**：系統在背景監聽觸發鍵，不影響其他操作

### 3. 🎒 一鍵清包功能
- **UI截圖記錄**：點擊「📸 截取背包UI」按鈕記錄背包介面
- **顏色基準設定**：點擊「🎨 記錄淨空顏色」設定背包淨空時的顏色
- **智能偵測**：系統自動偵測哪些格子有物品（基於顏色差異）
- **安全檢查**：F3執行前會檢查背包UI是否可見，避免誤操作
- **右鍵中斷**：按住右鍵2秒可中斷清包過程

### 4. 🖱️ 自動連點工具
- **AHK整合**：支援AutoHotkey腳本(.ahk)和編譯版(.exe)
- **自動啟動**：程式啟動時自動啟動AHK腳本
- **熱鍵控制**：使用CTRL+左鍵啟動自動連點（50ms間隔）
- **安全退出**：程式關閉時AHK腳本自動停止
- **管理員權限**：自動以管理員權限運行確保穩定性

### 5. 🔍 環境檢查工具
- **依賴項驗證**：自動檢查所有必要的Python套件
- **系統環境**：檢查作業系統版本和系統資訊

## ⚙️ 設定教學

### 血量/魔力區域設定
1. **選擇區域**：點擊「📦 框選血量區域」或「🔵 框選魔力區域」
2. **框選範圍**：在遊戲畫面中拖拽選擇監控區域
3. **確認設定**：選擇完成後會自動儲存座標
4. **顏色調整**：根據遊戲畫面調整HSV參數以獲得最佳檢測效果

### 觸發條件設定
1. **新增觸發**：
   - 類型：選擇「血量」或「魔力」
   - 百分比：設定觸發的百分比閾值（例如：70%、30%）
   - 快捷鍵：設定要觸發的按鍵（支援單鍵和多鍵序列）
   - 冷卻時間：設定冷卻間隔毫秒數（預設1500ms）

2. **多重設定範例**：
   ```
   血量 90% → 按鍵 1 (生命藥水)
   血量 60% → 按鍵 2 (緊急藥水)
   魔力 30% → 按鍵 3 (魔力藥水)
   ```

### 技能連段設定
1. **選擇套組**：選擇要設定的連段套組（1-3個）
2. **觸發鍵設定**：選擇觸發鍵（Q、W、E等）
3. **連段序列**：設定1-5個技能按鍵
4. **延遲時間**：設定每個連段之間的延遲（100-500ms）
5. **啟用套組**：勾選要使用的連段套組

### 一鍵清包設定
1. **UI截圖**：點擊「📸 截取背包UI」記錄背包介面
2. **顏色基準**：點擊「🎨 記錄淨空顏色」設定淨空顏色
3. **格子調整**：微調背包格子的位置偏移
4. **測試執行**：按F3測試清包功能

## 🔧 故障排除指南

### 常見問題解決

#### 血魔監控問題
1. **檢測不準確**：
   - 重新設定監控區域，確保顏色純淨
   - 調整HSV參數以適應不同顯示設備
   - 檢查遊戲畫面是否有特效或濾鏡干擾

2. **觸發失效**：
   - 確認遊戲視窗處於前台焦點
   - 檢查快捷鍵設定是否正確
   - 驗證冷卻時間設定是否合理

#### 技能連段問題
1. **連段不執行**：
   - 確認觸發鍵設定正確
   - 檢查連段序列是否完整
   - 驗證延遲時間是否適當

2. **按鍵衝突**：
   - 避免使用遊戲原生快捷鍵
   - 檢查是否有其他程式佔用相同按鍵
   - 調整延遲時間避免按鍵重疊

#### 一鍵清包問題
1. **無法清包**：
   - 確認已記錄背包UI截圖
   - 檢查淨空顏色基準是否正確設定
   - 驗證遊戲視窗處於正確狀態

2. **清包不完整**：
   - 調整背包格子偏移參數
   - 重新記錄背包UI截圖
   - 檢查顏色偵測閾值設定

#### 系統問題
1. **程式啟動失敗**：
   - 以管理員權限運行
   - 檢查Python環境是否正確安裝
   - 驗證所有依賴項是否完整

2. **效能問題**：
   - 降低監控更新頻率
   - 關閉不必要的預覽功能
   - 檢查系統資源使用情況
