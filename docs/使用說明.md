# PathofExile Sid 遊戲血量監控工具 使用說明

## 📋 工具簡介

PathofExile Sid GameTools_HealthMonitor 是一款專為 Path of Exile（流亡黯道）遊戲設計的自動化血量魔力監控工具。該工具能夠實時監控遊戲角色的血量與魔力狀態，並在設定條件下自動觸發對應的快捷鍵動作，提升遊戲體驗和反應速度。

## 🚀 快速開始

### 環境需求
- Windows 10/11 操作系統
- Python 3.10 或更高版本
- 以管理員權限運行（推薦）

### 安裝步驟
1. **環境安裝**：雙擊運行 `環境安裝.bat`，自動安裝Python和所需依賴包
2. **啟動工具**：雙擊運行 `啟動工具.bat` 或直接運行 `GameTools_HealthMonitor.exe`

## 🎮 主要功能

### 1. 血量監控
- **即時檢測**：自動識別遊戲中的血量條顏色變化
- **精確計算**：將血量轉換為0-100%的百分比顯示
- **視覺預覽**：提供即時的血量條預覽圖片

### 2. 魔力監控
- **藍色檢測**：識別魔力條的藍色像素
- **百分比計算**：精確計算魔力百分比
- **同步顯示**：與血量監控同步更新

### 3. 自動觸發系統
- **多重觸發**：支援設定多個血量/魔力觸發點
- **快捷鍵綁定**：每個觸發點可綁定不同的鍵盤按鍵
- **冷卻控制**：防止頻繁觸發的冷卻時間機制
- **優先級排序**：低百分比設定優先觸發

### 4. 視窗檢測
- **自動尋找**：自動檢測 Path of Exile 遊戲視窗
- **動態調整**：支援多顯示器和不同解析度
- **即時截圖**：對指定區域進行即時截圖分析

## ⚙️ 設定教學

### 血量/魔力區域設定
1. **選擇區域**：點擊「設定血量區域」或「設定魔力區域」
2. **框選範圍**：在遊戲畫面中拖拽選擇監控區域
3. **確認設定**：選擇完成後會自動儲存座標

### 觸發條件設定
1. **新增觸發**：
   - 類型：選擇「血量」或「魔力」
   - 百分比：設定觸發的百分比閾值（預設60%）
   - 快捷鍵：設定要觸發的按鍵（預設1）
   - 冷卻時間：設定冷卻間隔毫秒數（預設1500ms）

2. **多重設定範例**：
   ```
   血量 90% → 按鍵 1 (生命藥水)
   血量 60% → 按鍵 2 (緊急藥水)
   魔力 30% → 按鍵 3 (魔力藥水)
   ```

### 顏色檢測調整
1. **顏色閾值**：調整紅色（血量）和藍色（魔力）的檢測敏感度
2. **HSV參數**：進階使用者可調整色相、飽和度、明度範圍
3. **預覽測試**：即時查看檢測效果並調整參數

## 🎯 使用技巧

### 最佳設定建議
1. **血量區域**：選擇血量條最純紅色的區域
2. **魔力區域**：選擇魔力條最純藍色的區域
3. **觸發設定**：建議設定多個不同百分比的觸發點
4. **冷卻時間**：根據藥水持續時間調整冷卻間隔

### 常見問題解決
1. **檢測不準**：重新設定監控區域，確保顏色純淨
2. **觸發失效**：檢查授權狀態和快捷鍵設定
3. **性能問題**：調整更新間隔或關閉預覽功能
4. **視窗未找到**：確保遊戲以視窗模式運行

### 性能優化
- **更新間隔**：根據電腦性能調整監控頻率
- **預覽控制**：不需要時關閉預覽功能
- **後台運行**：最小化到系統托盤減少資源佔用

## 🔧 故障排除

### 常見錯誤
1. **Python未安裝**：運行 `環境安裝.bat` 重新安裝
2. **權限不足**：以管理員身份運行工具
3. **遊戲未檢測**：確認遊戲視窗標題設定正確
4. **功能異常**：重新啟動工具或檢查設定檔案

### 效能問題

- 關閉不需要的預覽功能
- 檢查系統資源使用情況
- 確保防毒軟體未阻擋
