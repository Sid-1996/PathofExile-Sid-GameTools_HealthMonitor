📖 關於 / About

本專案是以 影像辨識 (OpenCV) 與 輸入模擬 為核心技術的 Path of Exile 自動化研究工具，提供即時血量/魔力監控、技能連段、背包自動化清包取物、 自動連點及技能計時器等功能。所有操作均為外部圖像辨識，不修改遊戲檔案、不注入程式碼、不讀取遊戲記憶體。

Built with Python + OpenCV + AutoHotkey. This is an external automation research toolkit for Path of Exile. It uses image recognition and input simulation — no code injection, no memory reading.

✨ 功能總覽 Features

🩸

血量 / 魔力監控 Health / Mana

即時百分比判定、多閾值觸發、冷卻保護。支援低血警告與自動回復觸發。

⚡

技能連段 Skill Combo

自訂技能序列、自訂延遲、多組配置。毫秒級精確執行。

🎒

背包自動化 Inventory

一鍵清倉整理、固定點取物、UI 顏色辨識、左/右鍵點擊模式切換。

🖱️

自動連點 Auto Clicker

Ctrl + Left Click 切換開關，背景穩定運行。

⏱️

技能計時器 Skill Timer

自訂技能槽位、毫秒級精度間隔、支援單鍵與組合鍵。

🔍

背包預覽 Preview

即時遊戲畫面截圖預覽，支援排除標記（藍色疊加層），一鍵切換。

🌐

多語言支援 Multi-language

內建繁體中文與英文語言包，可擴充翻譯。

🛡️

視窗焦點感知 Activation-Aware

遊戲視窗失去焦點時自動暫停監控，避免錯誤觸發。

⌨️ 熱鍵一覽 Hotkeys
