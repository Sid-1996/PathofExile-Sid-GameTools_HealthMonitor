# 快速開始指南

## 🎮 PathofExile Sid GameTools_HealthMonitor

歡迎使用智能血量魔力監控工具！

### 📋 包裝內容

此壓縮包包含：
- `GameTools_HealthMonitor.exe` - 主程式
- `auto_click.exe` - 輔助工具
- `環境安裝.bat` - 環境安裝腳本
- `啟動工具.bat` - 啟動腳本
- `使用說明.md` - 詳細使用說明

### 🚀 三步快速安裝

#### 步驟 1: 安裝環境
1. 右鍵點擊 `環境安裝.bat`
2. 選擇「以系統管理員身分執行」
3. 等待 Python 和相關套件安裝完成

#### 步驟 2: 啟動程式
1. 雙擊 `啟動工具.bat` 或直接運行 `GameTools_HealthMonitor.exe`
2. 程式會自動初始化所有功能
3. 完整功能立即可用

#### 步驟 3: 基本設定
1. 點擊「設定」按鈕
2. 使用滑鼠框選血量和魔力條區域
3. 調整檢測參數並儲存
4. 開始享受智能監控！
